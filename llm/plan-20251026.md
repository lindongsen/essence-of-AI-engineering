# 任务目标

在个人电脑上训练一个简单的LLM，以知识库为素材进行验证，输出真实验证报告。

- 工作空间：/林生的奇思妙想/llm
- 项目名：llm1026

## 任务定义

你是一个轻量级中文语言模型，专注于理解和生成自然的中文文本。
你的训练目标是掌握基础的语言规律、常见问答、日常对话和简单文本生成。
保持中立、友好、简洁的表达风格，避免复杂术语。

## 完成条件

1. 所有单元测试通过；
2. 损失值小于0.03；
3. 验证这个问题，要得到正确的答案：
```
问：T-ARA的出道作品是什么？
答：《谎言》
```

4. 如果训练效果差，允许任务失败。

## 知识库

使用写作助手，执行`{工作空间}/text.md`的任务，你要把此文件路径告知写作助手。

## 技术栈

- 使用uv管理项目，`uv init`初始化项目，`uv add`增加依赖包，`uv run`执行命令行；
- 中文语言模型：uer/gpt2-chinese-cluecorpussmall；
- 中文分词器：google-bert/bert-base-chinese；
- 模型微调：LoRA；

## 实现细节

- 要提供一个脚本去实现问答，脚本参数可以指定要使用的模型；
