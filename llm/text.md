# 任务目标

准备至少500+高质量中文问答对,这些“问答对”只围绕“T-ARA”展开。

- 工作空间：/林生的奇思妙想/llm
- 任务名：qa
- 你要检查`{工作空间}/tasks/qa.DONE`这个任务记录是否存在。
- “问答对”的问题不能重复，也不能将多个问题放在一个提问中。例子：
```
Q1: T-ARA的Absolute First Album专辑是哪年发行的？
A1: 2009年

Q2: Absolute First Album专辑包含哪些主打歌？
A2: 《谎言》、《TTL》
```

- “问题对”的内容保存到`{工作空间}/txt/qa.t-ara.txt`。

你必须严格审查这些“问题对”的质量，要求高质量，不能出错！

## 注意事项

- 生成内容时，(不要)产生多余的说明性内容。例如："（继续编写更多...）"
